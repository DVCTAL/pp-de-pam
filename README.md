#  Previsão do Tempo

Aplicação web de **previsão do tempo** que permite ao usuário consultar as condições climáticas informando um **CEP**.

O sistema exibe a previsão atual e dos próximos dias, incluindo:
- Condição do tempo
- Temperatura máxima e mínima
- Ícones representativos do clima

##  Funcionalidades

-  Busca de previsão do tempo pelo **CEP**
-  Exibição da condição climática (chuva, sol, nublado, etc.)
-  Temperaturas **máxima e mínima**
-  Previsão para vários dias

---

##  Tecnologias Utilizadas

- .dotnet MAUI
- API de previsão do tempo
